# 🌀 Auto Reload

Webページ上に表示されるトグルUIから、ページの自動リロードを簡単にON/OFFできるChrome拡張機能です。オプションページでリロード間隔のカスタマイズも可能です。

## ✅ 主な機能

- ページ上にフローティングUI（トグルスイッチ）を表示し、クリックで自動リロードのON/OFFを直感的に切り替えできます。
- フローティングUIは、ドラッグ＆ドロップでページ内の好きな位置に移動可能です。
- 拡張機能アイコンのクリックで、タブごとに機能全体のON/OFFを制御できます。
- ONの時はアイコンに「ON」バッジが表示され、フローティングUIが利用可能になります。
- OFFの時はUIが非表示になり、自動リロードも停止します。
- オプションページから、自動リロードの間隔を秒単位で自由に設定可能です。

## 🧩 導入方法

1. 本拡張機能のソースコード（フォルダ全体）を、ダウンロードまたはクローンし、任意の場所に保存します。
2. Chromeブラウザを開き、アドレスバーに`chrome://extensions/`と入力して拡張機能管理ページにアクセスします。
3. ページ右上にある「デベロッパーモード」のトグルスイッチをONにします。
4. 左上に表示される「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします。
5. ファイル選択ダイアログが表示されるので、手順1で保存した拡張機能のフォルダ（`manifest.json`ファイルが含まれるフォルダ）を選択します。
6. 拡張機能がリストに追加され、ツールバーにアイコンが表示されれば導入完了です。

## ⚙️ 使い方

### 🧲 拡張機能のON/OFF（ツールバーアイコンから

- Chromeのツールバーに表示されている本拡張機能のアイコンをクリックします。
- クリックするたびに、現在アクティブなタブに対して拡張機能のON/OFFが切り替わります。
  - ONの場合: アイコンに「ON」というバッジが表示され、ページ上にフローティングUIが表示されます（表示位置は前回終了時の位置、またはデフォルト位置）。
  - OFFの場合: アイコンのバッジが消え、ページ上のフローティングUIも非表示になり、もし自動リロードが実行中であれば停止します。
- このON/OFF状態はタブごとに独立して管理されます。ページをリロードしても、この状態（およびバッジ表示）は維持されます。

### 🔁 自動リロードのON/OFF（ページ上のフローティングUIから）

- 拡張機能がONの状態で、ページ上に表示されている「🔁 Auto Reload」のフローティングUI内のチェックボックスを操作します。
  - チェックを入れる: 設定された間隔（デフォルトは10秒）でページの自動リロードを開始します。
  - チェックを外す: 自動リロードを停止します。
- フローティングUIの「≡」部分をドラッグすることで、UIをページ内の好きな位置に移動できます。
- 拡張機能をツールバーアイコンからONにした直後は、このチェックボックスは常にOFFの状態で始まります。
- このチェックボックスの状態とUIの表示位置は、ページをリロードしても維持されます（タブを閉じるかブラウザセッションが終了するまで）。

### ⏱️ リロード間隔の変更（オプションページから）

1. 以下のいずれかの方法でオプションページを開きます。
  - 拡張機能管理ページ (`chrome://extensions/`) で、本拡張機能の「詳細」をクリックし、「拡張機能のオプション」を選択します。
  - ツールバーの拡張機能アイコンを右クリックし、「オプション」を選択します。
2. 「リロード間隔（秒）」の入力フィールドに、希望するリロード間隔を秒数で入力します（例:`5`で5秒間隔）。1秒以上の値が設定できます。
3. 「保存」ボタンをクリックすると、設定が保存されます。
  - この設定はChromeアカウントで同期され、同じアカウントでログインしている他のデバイスにも反映されます。

## 📝 ファイル構成

```
auto-reload-extension/
├── README.md           (このファイル)
├── icon.png            (拡張機能アイコン)
├── manifest.json       (拡張機能の定義ファイル)
├── background.js       (バックグラウンド処理、状態管理)
├── content.js          (ページに挿入されるUI制御、リロード処理)
├── toggle.css          (フローティングUIのスタイルシート)
├── options.html        (オプションページのHTML)
└── options.js          (オプションページのJavaScript)
```

## 🛠 備考

- フローティングUIのチェック状態（リロードON/OFF）と表示位置は、`sessionStorage`を使用してタブごとに保持されます。これらは、タブを閉じるかブラウザのセッションが終了するまで維持され、ページのリロードでは消えません。
- 拡張機能全体のON/OFF状態（ツールバーアイコンでの操作）は、`background.js`内の変数でタブごとに管理され、ページリロードやブラウザ再起動後も（タブが復元されれば）維持されることを目指しています。
- リロード間隔の設定は`chrome.storage.sync`を使用しているため、同じGoogleアカウントでChromeにログインし同期を有効にしている場合、異なるデバイス間でも設定が共有されます。
- 本拡張機能は、`http://`または`https://`で始まるウェブページで動作します。Chromeの内部ページ（`chrome://`など）やローカルファイル（`file://`）では動作しません。
